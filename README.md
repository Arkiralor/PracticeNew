<p>
A repository to practice algorithms in python and to try out coding challenges.

## Setup

1. `python -m venv env`
2. `source env/bin/activate`
    - `source env/Scripts/activate` on Windows.
3. `pip-compile`
4. `python -m pip install -r requirements.txt`
